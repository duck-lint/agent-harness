## 1. Layer-0 — Seed Axioms|Layer-0 | 1. Layer-0 — Seed Axioms|Seed Axioms
### Seed Axiom F | Finitude / Asymptote
>A knower is a **Time-bounded, Attention-bounded** *indexical process* operating under *public constraints*.

#### Forbids
- World-Scope closure (“this is complete / final / total”) without quarantine.
- Treating confidence as unlimited budget (infinite ∞ Attention / infinite ∞ Time).
- Freezing a live model into a Dogma|dogmatic Identity badge (anti-update).

#### Common failure mode
- Scope/completeness inflation; budget denial; pointer-closure mistaken for world-closure.

### Seed Axiom R | Register Integrity
>There are **two irreducible registers** that may not be eliminated without Category Error:
>1. **Agent-independent constraints (Public)**
*(Phenomena and regularities that do not depend on any agent’s internal state; models of these (e.g., “laws”) belong to representation, not this register)*
>1. **Indexical presence**
*(“I / here / now / what-it’s-like” + commitment Space)*

#### Forbid
- Untyped moves (no Register declared).
- Cross-Register export/import without an explicit Inferential Bridge (Rule)|inferential bridge rule.
- Total reduction of one Register into the other with zero remainder.
- “Mixed” typing presented as a third Register (it’s usually just untyped crossing).
- Treating Provenance / source-access as if it could override Register discipline.

#### Common failure mode
- _Metaphysics smuggling_: a subverted Register mix gets treated as successfully crossed and “obvious” (often phenomenal).

### Seed Axiom U | Unification precedence
>Nothing is available for analysis unless it has already been unified (synthesized) in Experience.

#### Forbids
- Treating analysis as if it can access a “raw manifold” prior to unity.
- Confusing reverse-engineering with inversion (thinking you can uniquely recover the manifold as something in-itself to behold).
- “*Definition-first Metaphysics*” that forgets **synthesis is doing the enabling work**.

#### Common failure mode
- _Biting Your Own Teeth_: trying to observe the constituting operation *as if it were just another Object in the same Register*.

### Corollary | Inference Lifeline (Hard Typing → Transcendental Discipline)
1. **Inference dies without hard typing**  
   → structural constraint (Logic / Register integrity)
2. **Hard typing keeps inference alive under finitude**  
   → what makes inference possible *while* the agent exists
3. **Inference made from finitude does not survive death**  
   → Temporal bound on epistemic authority (no posthumous “infinite budget”)
Implications:
- Untyped claims do not merely “lack rigor”; they fail to qualify as inferential moves.
- Epistemic authority is horizon-bounded: it depends on a living ability to answer challenges, revise, and supply procedures.
- “Closure” is always a typed artifact (Scope, budget, quarantine), never a world-capture.

## 4. Layer-M — Meta-Ontology|Layer-M | 4. Layer-M — Meta-Ontology|Meta-Ontology
### Purpose
- 4. Layer-M — Meta-Ontology|Layer-M specifies presuppositions and constraints of intelligibility (manifold-as-given → synthesis → Object-unity). It functions as the ground for typing + linting that prevents illegitimate metaphysical reification and cross-Register smuggling or coercion.

### Non-goals
- **Not** an Ontology of “what ultimately exists.”
- **Not** a causal model of cognition.
- **Not** a mechanism that recovers world-Scope completeness from inside finitude.

### Core primitives
#### The 'move' (an atomic unit):
- Is any _(necessarily) typed **act of intelligibility**_ (perceptual “taking-as,” conceptual grouping, judgment, report, commitment, directive). Linguistic “claims” are one surface *Form* of moves.

#### Typing axes (orthogonal, not a register hierarchy)
- Every material move should be typed across three axes:
  1. **Register**: what kind of target is being claimed?
	  - Public: agent-independent constraints, invariants, shareable targets.
	  - Indexical: first-person givenness, attention, finitude, commitments.
  2. **Provenance / evidence state**: how did this move enter the system?
	  - Observed artifact: text, code, schema, trace, dashboard, model, or note as Representation / probe.
	  - Observed runtime behavior: test result, execution output, measurement, experiment, or direct operational trace.
	  - User report: stated goal, preference, experience, remembered behavior, or commitment.
	  - External source: literature, media, Art, experiment, citation, or source bundle with source metadata.
	  - Inference: derived from typed sources by a declared method.
	  - Speculation / unknown: quarantined possibility without adequate source support.
  3. **Action state**: what is being done with the move?
	  - Described / diagnosed.
	  - Proposed / recommended.
	  - Implemented / changed.
	  - Validated / invalidated.
	  - Deferred / quarantined.
- Provenance is **not** a third Register. It is a meta-typing axis for source-access. It can categorize any Register-typed move by asking how the move is known, reported, inferred, or merely suspected.
- Register answers: *what kind of claim-target is this?* Provenance answers: *what access path supports it?* Action state answers: *what lifecycle state does it occupy inside the system?*
- No axis transfers authority by itself. A Public-typed claim with weak Provenance remains weak; an Indexical report with sincere provenance does not become Public world-talk without an Inferential Bridge (Rule)|inferential bridge.

#### Registers (base types):
- **Public (non-indexical):** shareable targets, public invariants, *Belief-independent constraints*.
- **Indexical (first-person):** Attention/Embodiment/finitude constraints, lived givenness, commitment Space (I/me/what-it's-likeness).

##### Modes (not extra registers)
- Descriptive (Intrinsic to Public)
- Experiential report (Intrinsic to Indexical)
- Commitment (emergent): coordination-directives justified via public constraints + explicit bridges + explicit commitments

#### Inferential Bridge (Rule)|Inferential Bridge:
- An explicitly constrained rule that licenses translation between non-identical registers while *preserving intelligibility* and **tracking loss** (measurement, operationalization, report protocol, model mapping, etc.). Without this, an intelligible inference cannot be made.

#### Anti-reification Principle:
- Not treating Representation, or a product thereof (Object-unity (the resulting synthesis from Manifold of Perception) / Concept / Model / Object) as ontically closed or world-complete in-itself; reification is a violation of Register integrity without declaring the Inferential Bridge (Rule)|inferential bridge, **in whole** (not just what tracks, also what is lost).

#### Seed axiom jurisdiction
- This layer, 4. Layer-M — Meta-Ontology|Layer-M does not “prove” the 1. Layer-0 — Seed Axioms|Seed Axioms rather, it defines the grounds of which they function as admissibility constraints. Full axiom statements live in Layer 0.

4. Layer-M — Meta-Ontology|Layer-M, through a Kantian transcendental discipline, governs **what kinds of claims are allowed to count as world-talk at all**, and **how valid claims may be inferred once admitted**. Explicitly split into two orthogonal checks: validity of inference (general Logic) and admissibility of world talk (transcendental Logic).

#### 1. Validity of Inference (VoI) — General Logic
**VoI answers:**  
> Given some premises, does this conclusion follow correctly?
- VoI is Register-agnostic. It does not ask whether premises are meaningful, grounded, or admissible as world-claims — only whether the inferential move is coherent.
Includes:
- Deductive validity
- Probabilistic Coherence (Bayesian updating; consistency of credences)
- Internal model Coherence
- “If X, then Y” correctness
Bayesian reasoning lives **here** as an update rule, not as a license to introduce hypotheses.
**Key point:**  
A conclusion can be perfectly valid under VoI and still be **illegitimate world-talk**.

#### 2. Admissibility of World-Talk (AoW-t) — Transcendental Logic
**AoW-t answers:**  
> Is this claim even eligible to be called “about the world” (Public-typed), given finite agents?
- AoW-t is a **typing judgment**, not an inference rule. It gates entry into the Public Register.
A claim is **admissible as Public/world-talk if and only if** it satisfies all of the following:

##### AoW-t Criteria (Finite-Agent Discipline)
1. **Operational Anchor**
   The claim specifies (explicitly or implicitly) an evidence-producing procedure.  
   *What would be done or observed?*
2. **Provenance Integrity**
	The claim declares its source-access path: observed artifact, runtime behavior, user report, external source, inference, or speculation / unknown.  
	*What kind of support is this, and what authority can it actually carry?*
3. **Intersubjective Transform**
   Another finite agent could, in principle, access the same evidential structure via lawful transformations  
   (measurement, translation, calibration, behavioral alignment, correction loops).
4. **Finitude Constraint**
   No appeal to omniscience, infinite precision, or unreachable “in-principle” cheats.  
   Possibility must be **procedurally approachable**, not merely conceivable.
5. **Register Integrity**
   No indexical Content (“for me,” “as experienced,” private access) is smuggled in while presenting the claim as world-level.
If any criterion fails, the claim is **ill-typed as world-talk**.  
It may still exist as Indexical, speculative, poetic, or heuristic — just not Public.

#### 3. Objectivity (Bayesianized, but Disciplined)
Within the Class of **AoW-t-admissible claims**, Bayesian reasoning applies:
- Competing models are compared via shared update rules.
- Evidence shifts posteriors.
- **Objectivity** tracks *posterior stability under admissible intersubjective transforms*.
Crucially:
- Bayesian updating **selects among** admissible models.
- AoW-t **defines the model Class itself**.
This blocks:
- “Omniscient observer” grounding (fails finitude).
- Metaphysical laundering of private intuitions into Public claims.
- Sci-fi bridges with no access path.

#### 4. Plurality and Objectivity (Transcendental Result)
Objectivity is constitutively intersubjective in Form:
- A claim counts as objective only if it is formulated as invariant under Perspective.
- Perspective-invariance has Content only relative to a plurality of possible finite subjects.
This plurality need not be actually instantiated at all times.
It must be pragmatically possible under finitude (reachable procedures, not cheats).

Animal communication is admissible.
Omniscience is not.
This is not Metaphysics — it’s discipline.
